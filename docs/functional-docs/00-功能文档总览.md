# 功能文档总览

产品名称：独立开发者项目机会雷达  
产品形态：微信小程序  
文档版本：v0.1  
日期：2026-06-01

## 文档目的

本文档集由 PRD 拆分而来，用于后续产品设计、研发排期、接口设计、测试用例编写和运营配置。每份文档聚焦一个功能域，避免所有需求堆在单一 PRD 中难以拆解。

## 文档目录

1. `01-公共功能.md`：底部导航、登录、机会卡、收藏、会员、额度、通用状态等跨页面能力。
2. `02-Tab1-找项目.md`：用户画像问卷、AI 项目推荐、推荐结果和机会卡入口。
3. `03-Tab2-机会榜.md`：榜单分类、榜单列表、榜单详情和内容运营机制。
4. `04-Tab3-市场雷达.md`：平台动态、竞品观察、市场信号和 AI 解读。
5. `05-Tab4-工作台.md`：项目对比、PRD 生成、MVP 清单、开发 Prompt 和验证计划。
6. `06-Tab5-我的.md`：收藏、历史记录、个人画像、会员中心和反馈投稿。
7. `07-AI与数据能力.md`：推荐引擎、内容生成、数据来源、评估策略和风控。
8. `08-收费与会员.md`：免费版、会员版、单次付费、额度和转化策略。
9. `09-推广与运营.md`：定位、内容栏目、渠道策略和冷启动路径。

## 5 Tab 信息架构

1. 找项目：核心入口，负责画像填写和 AI 推荐。
2. 机会榜：内容入口，负责项目灵感和用户回访。
3. 市场雷达：市场证据入口，负责平台动态和竞品参考。
4. 工作台：转化入口，负责 PRD、MVP、对比和验证计划。
5. 我的：资产入口，负责收藏、历史、会员和个人资料。

## MVP 功能边界

MVP 必须包含：

- 微信登录。
- 用户画像问卷。
- AI 推荐 3 个项目机会。
- 机会卡详情。
- 机会榜。
- 收藏。
- 简单 PRD/MVP 清单生成。
- 基础会员和单次付费。
- 市场雷达人工精选与半自动更新。

MVP 暂不包含：

- 全平台实时数据抓取。
- 完整 ASO 专业分析。
- 开发者社区和即时聊天。
- 项目众筹、融资或合伙人撮合。
- 自动生成完整可上线代码。
- 企业级 BI 数据看板。

## 核心转化链路

1. 用户通过内容或分享进入小程序。
2. 用户完成画像问卷。
3. 系统生成 3-5 个机会卡。
4. 用户查看机会卡详情。
5. 用户收藏、对比或生成 PRD/MVP。
6. 深度报告、完整 PRD、竞品对比触发付费。
7. 生成内容沉淀到“我的”，推动复访。

## 核心指标

- MVP 上线 30 天注册用户数：>= 1,000。
- 画像问卷完成率：>= 70%。
- 推荐机会卡详情点击率：>= 40%。
- 推荐机会卡收藏率：>= 20%。
- PRD/MVP 生成功能使用率：>= 15%。
- MVP 阶段付费转化率：3%-5%。
