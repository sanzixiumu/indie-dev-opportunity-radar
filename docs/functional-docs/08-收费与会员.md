# 收费与会员

## 功能定位

收费体系用于验证用户付费意愿，并覆盖 AI 生成、深度分析和数据运营成本。MVP 阶段应保持简单，优先验证用户愿意为“深度报告、完整 PRD、年费会员”付费。

## 免费版

目标：拉新和体验。

权益：

- 每日 3 次基础项目推荐。
- 查看部分机会榜。
- 每月 1 次简单 PRD。
- 收藏 10 个项目。
- 市场雷达仅查看摘要。

限制：

- 完整机会卡部分字段可折叠或限制查看。
- 深度报告需要单次付费。
- 收藏超过 10 个需要升级或删除旧收藏。

## Personal Pro

建议价格：29 元/月，199 元/年。

目标：主力付费档。

权益：

- 每日 20 次项目推荐。
- 完整机会卡。
- 完整机会榜。
- 每月 20 次 PRD/MVP 生成。
- 项目对比。
- 收藏不限。
- 查看市场雷达基础分析。
- 支持 Markdown 或 PDF 导出。

## Deep Plan

建议价格：69 元/月，499 元/年。

目标：面向真正准备启动项目的用户。

权益：

- 高额度项目推荐。
- 每月 100 次 PRD/MVP/对比生成。
- 深度竞品分析。
- 生成开发 Prompt。
- 生成冷启动方案。
- 生成商业模式分析。
- 支持项目持续跟踪。

## 单次付费

适合小程序场景，降低首次付费门槛。

- 单个深度项目报告：9.9 元。
- 单个完整 PRD：19.9 元。
- 单个竞品对比报告：19.9 元。
- 选题诊断包：29.9 元。

## MVP 推荐上线方案

MVP 阶段优先上线：

- 免费版。
- 9.9 元单次深度报告。
- 199 元年费会员。

暂缓过多复杂订阅档位，先验证用户愿意为什么结果付费。

## 付费触发点

推荐触发点：

- 用户查看完整机会卡。
- 用户生成完整 PRD。
- 用户生成深度项目报告。
- 用户生成竞品对比报告。
- 用户收藏达到免费上限。
- 用户想导出 Markdown 或 PDF。

## 额度规则

- AI 生成动作扣减额度。
- 普通浏览不扣减额度。
- 生成失败不扣减额度。
- 历史内容重复查看不扣减额度。
- 单次付费内容永久可在“我的”查看。

## 支付与订单

支付方式：

- 微信支付。

订单内容：

- 商品名称。
- 商品类型。
- 支付金额。
- 支付状态。
- 购买时间。
- 权益生效时间。
- 权益过期时间。

## 验收标准

- 免费、Pro、Deep 权益限制清晰生效。
- 用户可购买单次报告或会员。
- 支付成功后权益立即生效。
- 额度消耗和剩余额度展示准确。
- 生成失败不扣额度。
