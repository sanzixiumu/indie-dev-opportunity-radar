# Tab4 工作台

## 功能定位

“工作台”负责把项目机会转化为可执行文档和决策材料，是主要付费转化入口。核心能力包括项目对比、PRD 生成、MVP 清单生成、开发 Prompt 生成和验证计划生成。

## 页面结构

### 工作台首页

- 当前对比项目。
- 最近生成的 PRD。
- 最近生成的 MVP 清单。
- 生成工具入口。
- 会员额度提示。

### 项目对比

支持从找项目、机会榜、市场雷达和我的收藏加入项目。

对比规则：

- 最少 2 个项目。
- 最多 3 个项目。
- 超过 3 个时提示先移除一个项目。

对比维度：

- 开发难度。
- 预算。
- 人力。
- 变现速度。
- 获客难度。
- 竞品强度。
- 平台风险。
- 个人适配度。
- 推荐结论。

输出：

- 对比表。
- AI 总结。
- 推荐优先级。
- 适合先验证的项目。
- 不建议立即启动的原因。

### PRD 生成

输入：

- 机会卡。
- 用户画像。
- 用户补充说明。

输出结构：

- 项目背景。
- 目标用户。
- 用户痛点。
- 核心价值。
- 核心功能。
- 页面结构。
- 数据结构。
- 商业模式。
- 成功指标。
- 非目标。
- 风险和待确认问题。

### MVP 清单生成

输出结构：

- 第一版必须做什么。
- 可以暂时不做什么。
- 7 天验证计划。
- 14 天开发计划。
- 30 天上线计划。
- 验收标准。
- 冷启动动作。

### 开发 Prompt 生成

目标工具：

- Cursor。
- Claude Code。
- Codex。
- v0。

输出要求：

- 明确产品目标。
- 明确页面和功能范围。
- 明确技术约束。
- 明确数据结构。
- 明确验收标准。
- 明确暂不实现内容。

### 验证计划生成

输出结构：

- 落地页方案。
- 问卷问题。
- 种子用户访谈提纲。
- 冷启动渠道。
- 付费验证方式。
- 成功/失败判断标准。

## 权益与限制

### 免费用户

- 每月 1 次简单 PRD。
- 可查看基础对比结果。

### Pro 用户

- 每月 20 次 PRD/MVP 生成。
- 可使用项目对比。
- 支持 Markdown 或 PDF 导出。

### Deep 用户

- 每月 100 次 PRD/MVP/对比生成。
- 可生成深度竞品分析。
- 可生成开发 Prompt。
- 可生成冷启动方案。
- 可生成商业模式分析。

### 单次付费

- 单个完整 PRD：19.9 元。
- 单个竞品对比报告：19.9 元。
- 单个深度项目报告：9.9 元。
- 选题诊断包：29.9 元。

## 保存与导出

- 生成内容自动保存到“我的”。
- 用户可以重命名生成内容。
- Pro 和 Deep 用户支持导出 Markdown 或 PDF。
- 历史生成内容重复查看不扣减额度。

## 异常状态

- 未登录：引导登录。
- 额度不足：引导购买会员或单次付费。
- 对比项目不足：提示至少选择 2 个项目。
- AI 生成失败：允许重新生成，不扣减额度。
- 生成耗时较长：提示用户稍后在“我的”查看。

## 验收标准

- 用户可从机会卡一键生成 PRD。
- 用户可从机会卡一键生成 MVP 清单。
- 用户可选择 2-3 个机会卡进行对比。
- 生成内容自动保存到“我的”。
- 生成失败不扣减额度。
- 会员和单次付费限制生效。
