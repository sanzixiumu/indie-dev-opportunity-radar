# 公共功能

## 功能定位

公共功能覆盖所有 Tab 共享的基础能力，包括导航、登录、机会卡、收藏、加入对比、会员权益、额度控制、通用状态和基础数据模型。公共功能应优先稳定，因为后续各 Tab 都依赖这些能力。

## 底部导航

### Tab 顺序

1. 找项目
2. 机会榜
3. 市场雷达
4. 工作台
5. 我的

### 导航要求

- 底部导航固定展示 5 个 Tab。
- 当前 Tab 需要有明确选中态。
- 从机会卡、榜单详情、市场动态详情等二级页面返回时，应回到来源 Tab。
- 工作台和我的可承接从其他 Tab 发起的保存、生成、对比动作。

## 微信登录

### 触发时机

以下操作需要登录：

- 收藏机会卡。
- 加入项目对比。
- 生成 PRD。
- 生成 MVP 清单。
- 保存历史记录。
- 查看会员中心。
- 支付。
- 提交反馈或投稿。

### 登录策略

- 浏览榜单、市场雷达摘要、机会卡基础信息可以允许未登录访问。
- 用户触发资产保存或消耗额度的动作时，引导微信授权登录。
- 登录成功后继续执行登录前的原动作。

## 机会卡

### 功能定位

机会卡是产品最核心的内容单元。找项目、机会榜、市场雷达、工作台和我的都需要复用机会卡。

### 字段

- 项目名称
- 一句话简介
- 目标用户
- 核心痛点
- 推荐理由
- 市场证据
- 竞品参考
- 独立开发者切入点
- MVP 功能范围
- 预计开发周期
- 预算估算
- 技术栈建议
- 变现方式
- 获客渠道
- 风险提示
- 适配评分
- 标签
- 更新时间

### 操作

- 收藏
- 取消收藏
- 加入对比
- 移出对比
- 生成 PRD
- 生成 MVP 清单
- 生成深度报告
- 分享

### MVP 展示要求

- 列表态展示：项目名称、一句话简介、适配评分、开发周期、预算、标签、主要操作。
- 详情态展示：完整字段和工作台入口。
- 信息不足时展示“信息有限”，不得用确定语气补全未经验证的事实。

## 收藏

### 规则

- 免费用户最多收藏 10 个项目。
- Pro 和 Deep 用户收藏不限。
- 收藏内容保存在“我的收藏”中。
- 收藏按钮需要支持实时状态切换。

### 失败状态

- 未登录：引导登录。
- 免费用户超限：提示升级会员或删除旧收藏。
- 网络失败：提示稍后重试。

## 加入对比

### 规则

- 单次最多对比 3 个机会卡。
- 少于 2 个机会卡时不能生成对比报告。
- 对比项存入工作台。
- 用户可从找项目、机会榜、市场雷达、我的收藏发起加入对比。

### 失败状态

- 超过 3 个：提示先移除一个项目。
- 项目已存在：提示已加入对比。
- 未登录：引导登录。

## 会员与额度

### 权益类型

- 推荐次数。
- PRD 生成次数。
- MVP 清单生成次数。
- 对比报告次数。
- 深度报告次数。
- 市场雷达深度分析权限。
- 收藏数量限制。
- 导出权限。

### 通用规则

- 涉及 AI 生成的动作都需要扣减额度。
- 生成失败不扣减额度。
- 重复查看历史生成内容不扣减额度。
- 相同输入短时间内重复生成可优先复用缓存。

## 通用状态

### 加载态

- AI 推荐和内容生成需要展示明确进度提示。
- 超过 20 秒未完成时提供“稍后在我的记录中查看”的选项。

### 空状态

- 无收藏：提示用户去找项目或看机会榜。
- 无历史记录：提示用户生成第一份 PRD 或 MVP 清单。
- 无对比项目：提示从机会卡加入对比。

### 错误态

- 网络错误：允许重试。
- 额度不足：引导购买。
- 内容生成失败：允许重新生成。
- 数据过期：提示更新时间。

## 验收标准

- 5 个 Tab 可正常切换，二级页面返回路径正确。
- 未登录用户可浏览公开内容。
- 登录后可收藏、对比、生成和保存。
- 免费用户收藏超过 10 个时有明确提示。
- 对比项目数量限制为 2-3 个。
- 额度不足时能明确展示当前权益和购买入口。
