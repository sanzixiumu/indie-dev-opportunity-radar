# Tab5 我的

## 功能定位

“我的”负责个人资产沉淀、会员转化、历史记录和用户反馈。用户在其他 Tab 产生的收藏、对比、PRD、MVP 清单和生成记录，都应在这里集中管理。

## 页面结构

### 我的首页

- 用户头像和昵称。
- 当前会员状态。
- 剩余额度。
- 我的收藏。
- 我的项目对比。
- 历史生成记录。
- 我的画像。
- 会员中心。
- 反馈与项目投稿。

## 我的收藏

### 内容

- 收藏的机会卡。
- 收藏来源：找项目、机会榜、市场雷达。
- 收藏时间。
- 标签。

### 操作

- 查看详情。
- 取消收藏。
- 加入对比。
- 生成 PRD。
- 生成 MVP 清单。

### 限制

- 免费用户最多收藏 10 个项目。
- Pro 和 Deep 用户收藏不限。

## 我的项目对比

### 内容

- 当前对比项目。
- 历史对比记录。
- 对比报告。

### 操作

- 查看对比。
- 移除项目。
- 重新生成对比报告。
- 保存对比结果。

## 历史生成记录

### 记录类型

- PRD。
- MVP 清单。
- 深度项目报告。
- 竞品对比报告。
- 开发 Prompt。
- 冷启动方案。

### 字段

- 生成类型。
- 对应项目。
- 生成时间。
- 消耗权益。
- 内容状态。

### 操作

- 查看。
- 重命名。
- 删除。
- 导出。
- 再生成。

## 我的画像

### 内容

- 技能栈。
- 团队规模。
- 预算范围。
- 每周可投入时间。
- 偏好平台。
- 偏好市场。
- 变现偏好。
- 风险偏好。

### 操作

- 查看画像摘要。
- 编辑画像。
- 基于新画像重新推荐项目。

## 会员中心

### 内容

- 当前套餐。
- 到期时间。
- 剩余额度。
- 权益说明。
- 购买入口。
- 单次付费入口。
- 订单记录。

## 反馈与投稿

### 反馈类型

- 推荐不准。
- 项目信息错误。
- 希望增加平台。
- 希望增加榜单。
- 功能建议。

### 投稿类型

- 投稿项目机会。
- 投稿竞品案例。
- 投稿市场动态。

## 权限与隐私

- 用户画像、收藏、生成记录默认仅用户本人可见。
- 删除历史记录需二次确认。
- 用户支付信息通过微信支付处理，产品侧不保存敏感支付信息。

## 异常状态

- 未登录：展示登录引导。
- 无收藏：引导去找项目或机会榜。
- 无历史记录：引导生成第一份 PRD 或 MVP 清单。
- 会员过期：提示续费但保留历史资产查看权限。

## 验收标准

- 用户可以查看收藏、历史生成记录和当前会员权益。
- 用户可以修改画像并重新生成推荐。
- 生成内容在工作台完成后能同步出现在“我的”。
- 免费用户收藏上限生效。
- 用户可以提交反馈或投稿。
